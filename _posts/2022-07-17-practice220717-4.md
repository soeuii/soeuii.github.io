# 문제
자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.
예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

- N의 범위 : 100,000,000 이하의 자연수

# 풀이
## 1. 각 자릿수
자연수 N을 string으로 만들어서 한자리씩 가져온다. 
```javascript
for (let i=0; i<n.toString().length; i++){    
      
    }
```

## 2. 더하기
한자리씩 더한다. 
```javascript
let answer = 0;
    
for (let i=0; i<n.toString().length; i++){    
  answer += +n.toString()[i]
}

return answer;
```



# 전체 코드
```javascript
function solution(n)
{
    let answer = 0;
    
    for (let i=0; i<n.toString().length; i++){    
      answer += +n.toString()[i]
    }

    return answer;
}
```

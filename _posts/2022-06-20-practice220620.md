# 문제

이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다.
별(\*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

# 풀이

for문을 사용해서 가로길이만큼 \* 만들고 줄바꿈 후 열만큼 반복해준다.

```javascript
for (let i = 0; i < b; i++) {
  let answer = "";
  for (let j = 0; j < a; j++) {
    answer += "*";
  }
  console.log(answer);
}
```

## 전체

```javascript
process.stdin.setEncoding("utf8");
process.stdin.on("data", (data) => {
  const n = data.split(" ");
  const a = Number(n[0]),
    b = Number(n[1]);

  for (let i = 0; i < b; i++) {
    let answer = "";
    for (let j = 0; j < a; j++) {
      answer += "*";
    }
    console.log(answer);
  }
});
```

# 문제

자연수 n이 매개변수로 주어집니다. n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 return 하도록 solution 함수를 완성해주세요. 답이 항상 존재함은 증명될 수 있습니다.

- 3 ≤ n ≤ 1,000,000

# 풀이

x를 2부터 하나씩 늘리며 넣고 나머지가 1이 되면 그 x값을 반환한다.

# 전체 코드

```javascript
function solution(n) {
  let answer = 0;

  for (let i = 2; i < n; i++) {
    if (n % i === 1) {
      answer = i;
      break;
    }
  }

  return answer;
}
```
